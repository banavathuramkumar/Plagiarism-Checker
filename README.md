# 📄 Plagiarism Checker (LCS Algorithm)

The Plagiarism Checker is a web-based application that compares two documents and calculates their similarity using the Longest Common Subsequence (LCS) algorithm.


It includes:

- 🌐 Frontend (HTML, CSS, JavaScript)
- 🖥️ Backend (Node.js)





## ✨ Features

- 📂 Upload two documents
- ⚡ Analyze similarity instantly
- 📊 Visual similarity score (circular progress UI)
- 📖 Side-by-side document comparison
- 🔗 Backend support for processing


## 🧑‍💻 Usage
- Upload Document 1
- Upload Document 2
- Click Analyze Similarity
    - View:
      - 📊 Similarity Score
      - 📄 Compared documents
## 📊 Similarity Levels
- 0% – 40% → Low similarity
- 40% – 70% → Moderate similarity
- 70%+ → High similarity (Possible plagiarism)
